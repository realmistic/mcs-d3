###Quick start###
```
git clone https://github.com/zefirka/mcs-d3-boilerplate.git d3
cd d3
npm install
npm start
```

### Линтинг
```
npm run lint
```
Скрипт запускает JSCS и JSHint по всем ключевым .js файлам в репозитории.
